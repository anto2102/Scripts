# A repository of my scripts

> Set of Python scripts for Maya

---

## 🧩 Scripts Overview

| Script | Description | 🔗 |
|---|---|---|
| `NurbsCurveColorer.py`  | Easily add color to rig controllers, also work for mesh wireframe. | [📁](Maya/NurbsCurveColorer) |
| `SuperRigidSkinTool.py` | Helps with rigid skinning by apply uniform skin weights to selected vertices. | [📁](Maya/SuperRigidSkinTool) |

### ⚙️ Requirements

- **Autodesk Maya** 2024, 2025, or 2026 *( they might work on 2022 and 2023 as well )*
- **Python 3.x** (bundled with Maya 2022+)
