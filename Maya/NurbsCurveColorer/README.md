# Nurbs Curve Colorer

> Easily add color to rig controllers, also works for mesh wireframes.

---

## 🖼️ Preview

![Preview](/Maya/NurbsCurveColorer/NurbsCurveColorer_Preview.png)

## 🧩 Features

- Color using an index slider or an RGB picker
- Save up to 10 favorite colors
- Manage favorites
- Clear color in one click

## ⚙️ Technicalities

- The tool sets the **Object Display** > **Drawing Override** attributes of the shape (not the transform)
- It toggles **Enable Overrides** and sets the **Color** automatically
- Favorite colors are saved in option vars
